# TrainTicket
## 기차, 지하철 예매 프로그램

2017년에 cpp로 만들어 본 기차 예매 프로그램 입니다.

기차는 서울역, 대전역, 동대구역, 부산역까지 운행하며, 지하철은 4호선 당고개 ~ 오이도까지 운행합니다.

발행된 티켓은 파일로 저장 됩니다.


## 기능
1. 티켓 예매하기
2. 예약 전체 리스트 출력하기
3. 예약번호로 검색하기
4. 티켓을 파일에 출력하기
5. 종료
